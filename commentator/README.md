# Commentator Overlay

Full-screen roster overlay designed for the commentator booth.

Connects to CRG ScoreBoard and displays both teams' rosters in a readable layout so commentators can quickly reference skater names and numbers during the game.

## Installation

Copy this entire `commentator/` folder into your CRG installation:

```
<CRG root>/html/custom/view/commentator/
```

Then open in a browser:

```
http://<CRG-IP>:8000/custom/view/commentator/index.html
```
